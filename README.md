# BMI Calculator

A sleek and responsive **BMI (Body Mass Index) Calculator** built using HTML, CSS (Bootstrap + Custom), and JavaScript. This web app lets users calculate their BMI based on height and weight inputs, and provides visual feedback on their weight category.

---

##  Features

- Instant BMI calculation
- Clean, modern UI with Bootstrap 5
- Fully responsive design
- Validates user input
- Smooth animations for results
- Custom header image included

---

## Technologies Used

- HTML5
- CSS3 + Bootstrap 5
- JavaScript (Vanilla)
